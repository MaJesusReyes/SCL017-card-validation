¿Cómo funciona esta herramienta?

Esta página web verifica si el número ingresado (ya sea de tarjeta de crédito o débito) es válido mediante la utilización del Algoritmo de Luhn. Se define como una fórmula de suma de verificación diseñada para detectar cualquier error de un único dígito, así como casi todas las transposiciones de dígitos adyacentes.

ESTE SERVICIO ES SEGURO DE UTILIZAR

No accederemos ni almacenaremos ningún tipo de dato que ingreses en esta página web.
Todos los cálculos son realizados con tecnología JavaScript por lo que tu número de tarjeta de crédito se mantendrá a salvo contigo.
